## 18. Acceptance Criteria

Visual direction считается зафиксированным, если:

- concept name утвержден;
- main metaphor понятна;
- hero composition утверждена;
- palette определена;
- typography определена;
- cards/buttons/forms rules определены;
- references scored;
- forbidden styles перечислены;
- mobile rules определены;
- implementation spec можно писать без догадок.
