# 00_MASTER_WORKFLOW — Website Factory

## Цель
Производить премиум-сайты под ключ системно: от брифа и структуры до Next.js production, интеграций, аналитики и QA.

## Pipeline
1. Lead / Fit check
2. Client Discovery Brief
3. Site Blueprint
4. Copywriting System
5. Reference Pack
6. Hero Factory
7. Section Factory
8. Next.js Implementation
9. Motion / Interaction Pass
10. Technical Conversion Layer
11. QA Gate
12. Deploy / Handoff / Post-launch

## Роли инструментов
- GPT / Claude: стратегия, структура, copy, prompts.
- Stitch / visual tools: layout direction, concepts, mood.
- Refero / LazyWeb: референсы и паттерны.
- Codex / Cursor: production implementation.
- Vercel: preview / deploy.
- Supabase / Telegram / CRM: заявки, storage, notifications.
- Notion / Drive / GitHub: knowledge base, archive, versioned docs.

## Production rule
Сначала система. Потом визуал. Потом код. Потом интеграции. Потом QA. Потом клиент.

## Anti-patterns
- начинать с портрета или hero-картинки;
- принимать AI mockup как production UI;
- давать AI 20 референсов сразу;
- делать 10 hero-итераций без visual QA;
- показывать клиенту сайт без Gate.
