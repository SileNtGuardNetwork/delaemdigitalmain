# 02_SITE_BLUEPRINT_TEMPLATE — Website Factory

## Purpose
A reusable blueprint template for building high-conversion websites across niches.

The blueprint must be completed before design or code starts.

## Screen Template

```md
# Screen N — Name

## Goal
What this section must achieve commercially.

## User Emotion
What the visitor should feel or understand after this section.

## Main Copy
Kicker:
H2/H1:
Body:
Bullets:
CTA:

## Layout Pattern
The structural pattern: route map, diagnostic grid, split comparison, timeline, service cards, proof cards, founder editorial, FAQ accordion, intake form.

## Visual Direction
Mood, materials, colors, depth, image/asset usage.

## Motion
Reveal, hover, scroll, line drawing, transitions.

## Mobile Behavior
Stacking, order, compression, overflow risks, CTA position.

## Required Assets
Photos, screenshots, icons, diagrams, proof, legal data.

## Acceptance Criteria
What must be true before this section is accepted.

## Reject Criteria
What instantly disqualifies the section.
```

## Universal 13-Screen Structure

1. Hero — positioning, main offer, CTA.
2. Problem / Loss Map — where the customer loses money, leads, time, or trust.
3. Method / System — proprietary method or operating logic.
4. Offer / What We Build — what is included in the service/product.
5. Before / After — current chaos vs controlled outcome.
6. Process — how the work happens.
7. Products / Packages — clear ways to buy.
8. Proof / Cases — trust, examples, numbers, screenshots, artifacts.
9. Founder / Team / Authority — who owns the result.
10. Audit / Lead Magnet — low-friction first step.
11. Pricing / Scope — budget orientation and boundaries.
12. FAQ / Objections — remove friction.
13. Final CTA / Lead Form — conversion endpoint.

## 8-Screen Compact Version

1. Hero
2. Problem / Loss Map
3. Method + Offer
4. Before / After
5. Process
6. Proof + Authority
7. Packages + FAQ
8. Final CTA / Lead Form

## Quality Rule
Every section must move the visitor closer to a commercial action. If a section only decorates the page, remove or rebuild it.
