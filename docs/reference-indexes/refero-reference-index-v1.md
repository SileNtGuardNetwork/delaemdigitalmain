# Refero Reference Index v1

## Purpose

This index summarizes the responsibilities of the Refero reference set already synthesized in `docs/delaem-digital-design-bible-v1.md`. It does not copy raw Refero files and does not claim new research.

Use Refero for style, tone, rhythm, and visual behavior. Use LazyWeb for UX structure and pattern research.

## Reference Roles

### Linear

- **Take:** command discipline, compact surfaces, subtle borders, technical precision.
- **Do not take:** neon lime identity, developer-tool density, tiny UI everywhere.
- **Applies to:** ClientFlow System, Audit diagnostic rows, route/status logic, compact technical labels.

### Authkit

- **Take:** premium glass atmosphere, soft dark depth, frosted surfaces, blue atmospheric glow.
- **Do not take:** violet primary, login-product mood, glassmorphism overload.
- **Applies to:** hero atmosphere, final CTA surface, selective premium panels.

### Calendly

- **Take:** conversion clarity, obvious CTA hierarchy, form readability, mobile simplicity.
- **Do not take:** light theme, mass-SaaS friendliness, blue primary identity.
- **Applies to:** CTAs, pricing, final form, mobile flow clarity.

### Augen Pro

- **Take:** architectural restraint, negative space, sparse composition, precise placement.
- **Do not take:** light marble theme, tiny inaccessible display type, Apple-copy mood.
- **Applies to:** global spacing, fewer stronger elements, premium section rhythm.

### Mercury

- **Take:** cinematic founder hero, spacious dark rhythm, restrained executive authority.
- **Do not take:** fintech mood, mountain imagery, blue primary as brand.
- **Applies to:** hero, founder moments, premium first impression.

### SpaceX

- **Take:** high-stakes black cinematic confidence, mission-level contrast, minimal UI discipline.
- **Do not take:** rockets, space literalism, D-DIN copying, military/space brand tone.
- **Applies to:** hero confidence, launch-readiness tone, section drama restraint.

### xAI

- **Take:** AI minimalism, clean dark surfaces, pill/button discipline, input prominence.
- **Do not take:** sterile emptiness, weak CTA color, generic AI homepage mood.
- **Applies to:** hero restraint, form/input prominence, technical badges when needed.

### Fey

- **Take:** dashboard/data depth without clutter, near-black data surfaces, subtle elevation.
- **Do not take:** finance/trading literal visuals, chart overload, metric decoration.
- **Applies to:** Audit, ClientFlow diagnostics, system-readiness surfaces.

### Slash

- **Take:** luxury fintech trust, obsidian surfaces, copper/gold logic, high-ticket confidence.
- **Do not take:** banking literalism, serif luxury everywhere, excessive gold.
- **Applies to:** pricing, primary CTA, premium trust surfaces.

### dope.security

- **Take:** bold technical atmosphere, dark confidence, strong display moments.
- **Do not take:** violet primary, cyber-security tropes, cosmic effects overload.
- **Applies to:** occasional technical drama, atmospheric section accents, not core identity.

## Operating Rule

No single reference owns the site. Each reference contributes one job. Delaem Digital must remain its own system: dark, founder-led, architectural, controlled, high-ticket B2B, and ClientFlow-aware.
