# Product Architecture

## Definition

ClientFlow AI Quiz System v1 is an internal agency module for building premium diagnostic quizzes under different niches.

It is not a generic quiz builder. It qualifies leads, routes offers, creates sales context, sends lead cards, stores submissions and measures lead quality.

## Core model

One universal engine. Unique niche packs. Unique design packs. Shared analytics and integrations.

Included: config-driven quiz runtime, three first quizzes, Supabase storage, Telegram lead cards, webhook payloads, analytics events, consent/cookie baseline, anti-spam baseline and reporting format.
