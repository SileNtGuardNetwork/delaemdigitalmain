# Делаем Диджитал — Personal Site v0.1

Production-ready starter project for the personal website of Vitaly Timoshenko / «Делаем Диджитал».

The site demonstrates ClientFlow System: a controlled route from first touch to qualified request.

## Stack

- Next.js App Router
- TypeScript
- CSS design tokens
- Responsive layout
- Vercel-ready

## Core rules

- Do not change the hero offer without approval.
- Do not add cases or portfolio sections.
- Do not use “Делаем Заявки”.
- Do not use “ClientFlow X-Ray Pack”. Use “ClientFlow Аудит”.
- Do not use VT/DT as the main logo.
- Do not add stock people or generated replacement portraits.
- Do not add analytics, CRM or real form integrations in v0.1.
- Do not expose home/registration address in public marketing blocks.

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run build
npm run typecheck
```
